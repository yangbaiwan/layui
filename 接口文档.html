接口地址：ebank.dodou.cn/axWithdrawAPI.ashx

请求参数

名称 类型 描述 是否必填
WithdrawNum String 提现编号 Y
UserId String 商户Id Y
WithDrawMoney Int 提现金额（Int类型，单位：分） Y
BankName String 到账银行名称 Y
BankAccountName String 到账银行开户人 Y
BankCardNum String 到账银行卡号 Y
Remark String 提现备注 N
Sign String 签名：md5(WithdrawNum + UserId + WithDrawMoney + BankCardNum) Y

返回参数(Json)

名称 类型 描述
Success bool 请求是否成功
WithDrawMoney decilal 提现金额（decimal类型，单位：元）
WithdrawNum String 提现单号
BankName String 到账银行名称
BankAccountName String 到账银行开户人
BankCardNum String 到账银行卡号
ErrMsg String 错误信息
